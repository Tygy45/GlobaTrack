# OpenStreetMap версия - готова к использованию!

## Что работает:

✅ **Карта** - OpenStreetMap (бесплатная, без регистрации)
✅ **Поиск мест** - Nominatim API (адреса, улицы, заведения)
✅ **Маршруты по дорогам** - OSRM API (бесплатный роутинг)
✅ **Прямые линии** - встроенный расчет расстояний
✅ **Сохранение маршрутов** - работает как раньше
✅ **Просмотр в профиле** - все маршруты отображаются

## Запуск:

```bash
python app.py
```

Откройте: http://127.0.0.1:

## Как использовать:

1. **Поиск**: Введите адрес в поле справа вверху
   - Примеры: "Москва, Красная площадь", "Невский проспект", "Макдональдс Москва"

2. **Добавление точек**: Кликайте по карте

3. **Режим маршрута**: 
   - "Прямые линии" - прямое расстояние между точками
   - "По дорогам" - маршрут по реальным дорогам (OSRM)

4. **Сохранение**: Нажмите "Сохранить маршрут" и введите название

5. **Просмотр**: Откройте "Профиль" для просмотра сохраненных маршрутов

## Преимущества OpenStreetMap:

- Работает сразу, без API ключей
- Бесплатно и без ограничений для разработки
- Открытые данные, поддержка сообщества
- Маршруты строятся по реальным дорогам
- Автоматическая привязка к ближайшим дорогам

## Если не понравится:

Бэкап Яндекс.Карт сохранен в файле `templates/index_yandex_backup.html`
Инструкция по восстановлению в файле `RESTORE_YANDEX.md`

## Технические детали:

- **Leaflet.js** - библиотека для работы с картами
- **Nominatim** - геокодирование (поиск мест)
- **OSRM** - построение маршрутов по дорогам
- **OpenStreetMap** - картографические данные

Все работает через бесплатные публичные API!

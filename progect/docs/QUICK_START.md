# Быстрый старт

## Что исправлено:

✅ **Убран API ключ** - теперь карты работают без ключа (для разработки это нормально)

✅ **Поиск в header** - поле поиска находится справа вверху, работает для:
   - Адресов (Москва, Ленина 128)
   - Улиц (Невский проспект)
   - Заведений (Макдональдс, Пятерочка)
   - Достопримечательностей

✅ **Маршруты по дорогам** - используется multiRouter API для:
   - Автоматической привязки к ближайшим дорогам
   - Построения оптимального маршрута
   - Обработки ошибок (автопереключение на прямые линии)

## Запуск:

1. Убедитесь что Flask установлен:
```bash
pip install flask flask-sqlalchemy
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер: http://127.0.0.1:5000

## Как использовать:

1. **Поиск места**: Введите адрес в поле справа вверху и нажмите "Найти"
2. **Добавление точек**: Кликайте по карте
3. **Режим маршрута**: Переключайте между "Прямые линии" и "По дорогам"
4. **Сохранение**: Нажмите "Сохранить маршрут" и введите название
5. **Просмотр**: Откройте "Профиль" для просмотра сохраненных маршрутов

## Важно:

- Для работы нужен интернет (для загрузки Яндекс.Карт)
- Поиск работает на русском и английском языках
- Если маршрут по дорогам не строится, система автоматически переключится на прямые линии

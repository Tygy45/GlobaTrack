# Инструкция по настройке API ключа Яндекс.Карт

## Текущее состояние
Сейчас приложение работает **БЕЗ API ключа** - это нормально для разработки и тестирования.

## Если нужен API ключ (для продакшена)

### Шаг 1: Регистрация
1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите или зарегистрируйтесь

### Шаг 2: Получение ключа
1. Перейдите в раздел "JavaScript API и HTTP Геокодер"
2. Нажмите "Получить ключ"
3. Заполните форму (название проекта, описание)
4. Скопируйте полученный API ключ

### Шаг 3: Добавление ключа в код
Замените в файлах `templates/index.html` и `templates/profile.html`:

```html
<script src="https://api-maps.yandex.ru/2.1/?lang=ru_RU" type="text/javascript"></script>
```

На:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ_ЗДЕСЬ&lang=ru_RU" type="text/javascript"></script>
```

## Примечание
Для локальной разработки и тестирования API ключ НЕ обязателен. Яндекс.Карты работают без ключа с некоторыми ограничениями по количеству запросов.

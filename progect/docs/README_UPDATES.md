# Обновления сайта GlobeTrack

## Что было добавлено:

### 1. Построение маршрута по дорогам
- Добавлена кнопка переключения режима: "Прямые линии" / "По дорогам"
- В режиме "По дорогам" маршрут строится с учетом дорожной сети через Яндекс.Маршруты
- Автоматический расчет реального расстояния по дорогам

### 2. Поиск заведений и магазинов
- Добавлено поле поиска над картой
- Можно искать любые заведения, магазины, адреса
- При поиске автоматически создается метка на найденном месте
- Карта центрируется на найденном объекте

### 3. Отображение длины маршрута
- Блок "Информация о маршруте" показывает:
  - Длину маршрута в километрах
  - Количество точек на маршруте
- Автоматическое обновление при добавлении/удалении точек

### 4. Сохранение маршрута с названием
- При сохранении маршрута появляется запрос на ввод названия
- Сохраняется:
  - Название маршрута
  - Все точки маршрута
  - Длина маршрута
  - Режим построения (по дорогам или прямые линии)
  - Дата создания

### 5. Просмотр сохраненных маршрутов в профиле
- В профиле отображаются все сохраненные маршруты пользователя
- Для каждого маршрута показывается:
  - Название
  - Дата создания
  - Длина маршрута
  - Тип маршрута (по дорогам/прямые линии)
- Кнопка "Открыть маршрут" восстанавливает маршрут на карте
- Маршрут отображается в том же режиме, в котором был сохранен

### 6. Уменьшен размер карты
- Высота карты уменьшена с 500px до 400px
- Карта стала более компактной на главной странице

## Как использовать:

1. **Добавление точек вручную**: Кликайте по карте
2. **Поиск мест**: Используйте поле поиска в header (справа вверху). Можно искать:
   - Адреса (например: "Москва, Тверская 1")
   - Улицы (например: "Невский проспект")
   - Заведения (например: "Макдональдс Москва")
   - Достопримечательности (например: "Красная площадь")
3. **Переключение режима**: Нажмите кнопку "Режим: Прямые линии" для переключения на построение по дорогам
4. **Сохранение**: Нажмите "Сохранить маршрут" и введите название
5. **Просмотр**: Перейдите в "Профиль" и откройте нужный маршрут

## Особенности построения маршрута по дорогам:

- Если точка находится не на дороге, маршрут автоматически привязывается к ближайшей дороге
- Если построить маршрут по дорогам не удается (точки слишком далеко), система автоматически переключится на режим прямых линий
- Маршрут учитывает реальную дорожную сеть и строит оптимальный путь

## Технические детали:

- База данных обновлена (добавлены поля: route_name, route_length, use_roads)
- Используется Яндекс.Маршруты API для построения маршрутов по дорогам
- Используется Яндекс.Геокодер для поиска мест
